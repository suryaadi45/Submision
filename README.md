# Submision

## Setup Environment - Anaconda
```
conda create --name main-ds python=3.12.3
conda activate main-ds
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal
```
mkdir Submision
cd Submision
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard/dasboard.py
```
